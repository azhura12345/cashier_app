# Cashier App (POS)

A Point of Sale system built with PyQt5 and PostgreSQL.

## Features

- Browse products by category
- Add to cart & calculate orders
- Store transactions to PostgreSQL
- Modular UI layout using PyQt

## Requirements

- Python 3.8+
- PyQt5
- psycopg2
- python-dotenv
